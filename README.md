# 🛒 E-commerce Website User Logs Analysis

A comprehensive Data Analytics and Business Intelligence (BI) project focused on decoding user behavior patterns on an e-commerce platform. By analyzing web server logs (including IP addresses, page views, session durations, interaction paths, and user languages), this project uncovers actionable insights into user navigation and directly maps behavioral metrics to revenue and sales conversions.

## 🛠️ Tech Stack & Analytics Tools

* **Programming Language:** Python 🐍 (utilized in `Data_cleaning.ipynb` and `main.ipynb`).
* **Core Analytics Ecosystem:** Pandas, NumPy, Matplotlib, and Seaborn for statistical computing and data modeling.
* **Business Intelligence (BI):** **Microsoft Power BI** (utilized in `PowerBI_dashboard.pbix`) for building interactive visualizations, user activity trends, and geographic/demographic tracking via IP tracking.

---

## 🚀 Analytics Pipeline & Methodology

1. **Data Cleaning & Structuring:** Extracting raw fields from web log entries, cleaning unstructured behavioral metrics, and formatting temporal data (session lengths, timestamps).
2. **Exploratory Data Analysis (EDA):** Investigating user behavior patterns, detecting high-traffic navigation paths, analyzing demographic language preferences, and examining IP-based user interactions.
3. **Sales & Conversion Analysis:** Correlating session durations and interaction frequencies with successful checkouts and sales values to isolate high-value user behavior.
4. **Interactive Dashboard Development:** Designing an end-to-end Power BI executive dashboard enabling stakeholders to drill down into daily active user metrics, conversion rates, and revenue leakage points.
